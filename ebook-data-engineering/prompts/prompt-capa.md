# Prompt utilizado para gerar a capa

Compatível com geradores de imagem em inglês (MidJourney, DALL·E, Ideogram, etc.):

```
A futuristic data engineer working in a modern cyber security operations center,
holographic dashboards, Python code, SQL database, cloud computing,
glowing blue interface, artificial intelligence, professional ebook cover,
ultra realistic, cinematic lighting, 4k
```

## Ferramenta utilizada
MidJourney / Ideogram / DALL·E (ou outro gerador de imagens por IA).

## Observação
A imagem final deve ser salva em `assets/capa.png` e referenciada na primeira página do ebook.
