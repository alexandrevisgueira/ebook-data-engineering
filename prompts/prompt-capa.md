# Prompt utilizado para gerar a capa

## Prompt de imagem (fundo, sem texto)

Compatível com geradores de imagem em inglês (Nano Banana Pro, MidJourney, DALL·E, etc.):

A futuristic cyber security operations center with no people,
multiple holographic dashboards showing data visualizations, world map with
network connections, Python and SQL code on transparent glowing screens,
cloud computing icons, database symbols, glowing blue and purple interface,
empty modern control room, cinematic lighting, ultra realistic,
vertical portrait composition, no text, no words, no letters, no logos

## Ferramenta utilizada
Nano Banana Pro (Gemini) para geração da imagem de fundo.

## Observação importante
Geradores de imagem costumam "alucinar" texto (títulos, nomes de autor incorretos)
quando o prompt pede uma "capa de ebook" pronta. Por isso, o prompt gera apenas o
**fundo/cena**, sem nenhum texto. O título, subtítulo e nome do autor são adicionados
depois por cima da imagem (edição manual em Canva, PowerPoint, ou script Python/Pillow),
garantindo texto correto e legível.

A imagem final deve ser salva em `assets/capa.png`.
