# Prompt utilizado para gerar o texto do ebook

```
Você é um Engenheiro de Dados Sênior e escritor técnico.

Escreva um ebook profissional em português chamado:

"Data Engineering na Prática: 10 Boas Práticas de Segurança que Todo Iniciante Deve Conhecer"

O ebook deve possuir:
- Linguagem simples
- Exemplos reais
- Explicações objetivas
- Código quando necessário
- Dicas profissionais
- Conclusão

Cada capítulo deve possuir entre 300 e 500 palavras.

O público é composto por estudantes de Engenharia de Dados.

Utilize exemplos envolvendo:
Python, SQL, PostgreSQL, BigQuery, APIs, Docker, Cloud, GitHub

Sempre enfatize segurança:
.env, IAM, SQL Injection, Validação, Logs, Tratamento de erros, Segredos, Boas práticas.
```

## Ferramenta utilizada
ChatGPT / Claude (revisão técnica e ajuste de tom).
