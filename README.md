# 📘 Ebook - Data Engineering na Prática

Projeto desenvolvido para o desafio da DIO utilizando IA Generativa.

**Título:** Data Engineering na Prática: 10 Boas Práticas de Segurança que Todo Iniciante Deve Conhecer

## Sobre o projeto

Este ebook reúne, de forma direta e prática, os fundamentos de Engenharia de Dados
com foco em segurança — desde a arquitetura de um pipeline moderno até boas práticas
de proteção de credenciais, prevenção de SQL Injection e gerenciamento de secrets em nuvem.

## Ferramentas utilizadas

- ChatGPT / Claude — geração e revisão do texto técnico
- Gerador de imagens por IA — capa e ilustrações
- Canva / Word — diagramação

## Conteúdo

- O que é Engenharia de Dados e por que segurança é prioridade
- Arquitetura de um pipeline moderno (API → Python → Validação → PostgreSQL → BigQuery → Dashboard)
- 10 boas práticas de segurança (.env, .gitignore, SQL Injection, Logging, IAM, Secrets, etc.)
- Estudo de caso: Secure Marketing Data Pipeline
- Como ferramentas de IA (ChatGPT, Claude, Gemini, Copilot) aceleram o desenvolvimento
- Checklist profissional para projetos de dados
